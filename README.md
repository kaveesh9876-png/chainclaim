## ⚙️ Installation

### Clone Repository

```bash
git clone <your-repository-url>
cd chainclaim
```

### Install Dependencies

```bash
npm install
```

### Start Server

```bash
npm start
```

or

```bash
npm run dev
```

## 📡 API

| Method | Endpoint | Description |
|----------|----------|-------------|
| GET | /api/claims | Get all claims |
| POST | /api/claims | Create claim |
| PUT | /api/claims/:id | Update claim |
| DELETE | /api/claims/:id | Delete claim |
